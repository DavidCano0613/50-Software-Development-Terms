Software development dictionary

ALGORITHM
In computer science, programming, and math, an algorithm is a sequence of instructions where the main goal is to solve a specific problem
APPLICATION
Also known as an app, these are programs designed to perform functions. This includes mobile apps.
API
API is the acronym for Application Programming Interface, which is a software intermediary that allows two applications to talk to each other.
ADAPTIVE DESING
Adaptive design is a user interface that's adapted to different screen sizes.
BROWSER
a computer program with a graphical user interface for displaying and navigating between web pages.
BUG
A bug computer definition is referred to as a failure or a flaw in the software program.
BIT AND BYTES
A bit is a binary digit, the smallest increment of data on a computer. ... Bits are usually assembled into a group of eight to form a byte.
BACKEND
All of the behind-the-scenes digital operations that it takes to keep the front end of a website running, such as the coding, style, and plugins.
CODE
program instructions.
CMS
A content management system (CMS) is an application that is used to manage web content, allowing multiple contributors to create, edit and publish
CSS
Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML
CACHE
Cached data is information stored on your computer or device after you visit a website. Developers use cached data to improve your online experience. ... The goal of cached data is to speed up site loading.
COOKIES
Cookies are text files with small pieces of data — like a username and password — that are used to identify your computer as you use a computer network.
CRM
A CRM system essentially provides a central place where businesses can store customer and prospect data, track customer interactions, and share this information with colleagues.
DATA BASES
A database is an organized collection of structured information, or data, typically stored electronically in a computer system.
DOMAIN
Is the name of a website.
DevOps
DevOps is the combination of cultural philosophies, practices, and tools that increases an organization's ability to deliver applications and services at high velocity
DOCUMENTATION
Software documentation is written text or illustration that accompanies computer software or is embedded in the source code.
DEVELOPER
A developer is an individual that builds and create software and applications. He or she writes, debugs and executes the source code of a software application.
FRONT-END
The frontend of a software program or website is everything with which the user interacts
FRAMEWORKS
A framework is a particular set of rules, ideas, or beliefs which you use in order to deal with problems or to decide what to do.
FULL-STACK
A Full Stack Developer is someone who works with the Back End — or server side — of the application as well as the Front End, or client side.
GIT
is software for tracking changes in any set of files, usually used for coordinating work among programmers collaboratively developing source code during software development.
GITHUB
GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.
GUI
GRAPHICAL USER INTERFACE
HTTP
HTTP is a protocol for fetching resources such as HTML documents.
HTML
HTML (Hypertext Markup Language) is the code that is used to structure a web page and its content.
Internet
The Internet is a vast network that connects computers all over the world.
IP
An IP address is a unique address that identifies a device on the internet or a local network.
Java
Java is a technology consisting of both a programming language and a software platform
Javascript
JavaScript is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive.
KEYWORD
Keywords are the words and phrases that people type into search engines to find what they're looking for.
Machine Learning
the use and development of computer systems that are able to learn and adapt without following explicit instructions, by using algorithms and statistical models to analyse and draw inferences from patterns in data.
Navigation
With a computer, navigation refers to the act of opening and moving through computer menus, like the Start menu in Windows, opening software programs, or viewing files
Object
An object is an abstract data type with the addition of polymorphism and inheritance. Rather than structure programs as code and data, an object-oriented system integrates the two using the concept of an "object".
Parameter
A parameter is a named variable passed into a function. Parameter variables are used to import arguments into functions.
Plugin
A plugin is a piece of software that acts as an add-on to a web browser and gives the browser additional functionality.
Property
With programming, properties are special values included in a class/object, usually (depending on the language being used) set using get or set method calls.
Registers.
Registers are also called internal memory or immediate access memory stores.
Reserved word
In a computer language, a reserved word (also known as a reserved identifier) is a word that cannot be used as an identifier, such as the name of a variable, function, or label – it is "reserved from use".
Responsive desing
Responsive web design (RWD) is a web development approach that creates dynamic changes to the appearance of a website, depending on the screen size and orientation of the device being used to view it.
Requirements
The IEEE Standard Glossary of Software Engineering Terminology defines a requirement as: A condition or capability needed by a user to solve a problem or achieve an objective.
Server
A server is a computer connected to a network of other workstations called 'clients'. Client computers request information from the server over the network.
Semantics
Structured data
Structured data is data that adheres to a pre-defined data model and is therefore straightforward to analyse.
UI
At the most basic level, the user interface (UI) is the series of screens, pages, and visual elements—like buttons and icons—that enable a person to interact with a product or service.
UX
UX deals more generally with a user's overall experience with a brand, product or service.
Web
The World Wide Web (WWW), commonly known as the Web, is an information system where documents and other web resources are identified by Uniform Resource Locators (URLs, such as https://example.com/ ), which may be interlinked by hyperlinks, and are accessible over the Internet.
Widgets
Widgets can be added to your phone's home as a quick way to access certain information from apps without having to open the app itself.
